Play socket.io integration tests

This runs the Play socket.io integration tests.

You need to ensure that the Play TestSocketIOServer is running, this can be run by running sbt test:run. That server will serve this as the index file.

Enable JSONP tests

JSONP tests are disabled by default because they are slow and they make working with the tests harder because they turn the refresh button into a stop button, so you have to let all the tests run before you can hit refresh and try again.
